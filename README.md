# Rusty Tetris
Tetris implementation with Rust

## Features
 - [x] Single player mode
 - [x] Local versus Mode
 - [x] Score history and best scores tracking
 - [x] Persisting data with binary files
 - [x] Profile system
 - [x] Settings
 - [ ] Mouse support
